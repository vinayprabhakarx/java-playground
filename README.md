# Java Playground

A practice playground for learning, experimenting, and practicing Java programming.

## Purpose

This repository serves as a hands-on space to:

- **Practice** Java fundamentals and advanced concepts
- **Learn** new Java features and best practices
- **Experiment** with different libraries, frameworks, and coding patterns

## Getting Started

Clone or navigate to this directory and start exploring Java examples and exercises.

## Structure

All code is organized under the `com/vinayprabhakarx/` package. New topics and exercises will be added as separate directories within this structure.

```
com/
└── vinayprabhakarx/
    ├── basics/
    └── [more topics to be added...]
```

## How to Compile and Run

### Compile a Java file:

```bash
javac com/vinayprabhakarx/basics/part1/Exercise1.java
```

### Run a compiled Java class:

```bash
java com.vinayprabhakarx.basics.part1.Exercise1
```
